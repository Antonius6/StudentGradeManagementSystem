# Student Grades Management System 🎓

A simple **Java** program to manage student grades across three subjects.  
It calculates averages, assigns letter grades, and shows the highest grade per subject.

## 📋 Features
- Input student names and grades for 3 subjects.
- Calculate average grade for each student.
- Assign letter grades (A, B, C, D, F).
- Display the highest grade in each subject.
- Search for a student by name.
- Count how many students passed or failed.

## 🧠 How to Run
1. Open the project in **IntelliJ IDEA** or any Java IDE.
2. Run the `Main.java` file.
3. Follow the console instructions to enter data and choose from the menu.
